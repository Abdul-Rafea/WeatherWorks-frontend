# <img src="./frontend/src/assets/WasabiX_Logo.webp" width="75" alt="WeatherWorks Logo">WasabiX
> A full-stack weather app with social interactivity
**<sub>Version 0.6.5</sub>**

## Features
* Real-time weather insights from people right where you are.
* Weather alerts tailored to your exact neighborhood.
* Leave comments on curent weather forecast.
